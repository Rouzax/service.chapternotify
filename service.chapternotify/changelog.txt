Chapter Notify Changelog
========================

v0.3.0 (2026-04-07)
--------------------------
New Features:
  - Styled overlay notification for chapter changes during video playback.
    Designed for concert recordings and festival DJ sets with named chapters
    created by CrateDigger.
  - Chapter name parsing for "Artist - Track [Label]" format with split
    prefix/value labels. Prefixes stay fixed, values scroll for long text.
  - 6 overlay positions (top/bottom × left/center/right) with fade and slide
    animation styles.
  - EasyTV-style theming with 4 color themes: Golden Hour, Ultraviolet,
    Ember, and Nightfall. Accent border and separator line tinted to match.
  - Rounded panel background with configurable opacity (0-100%) and toggle
    to show/hide background entirely (floating text mode).
  - Configurable library paths (up to 3), display duration, and debug logging.
  - Test overlay button in addon settings.
  - Any button press dismisses overlay instantly. No input blocking.

Internal:
  - Layered background: white_rounded.png (dark panel, opacity-controlled) +
    border_rounded.png (hollow accent ring, full opacity on top).
  - WindowXMLDialog with quickclose property to skip closing animation on
    user dismiss, preventing action blocking during animation.
